React Touch UI

让你只专注于js业务逻辑处理

react 移动开发框架

专属于移动

Reac Touch UI 专为移动打造，在技术实现、交互设计上只考虑主流移动设备，适应微信、app内嵌H5、cordova 混合app开发场景。

专注于 UI

提供 UI 组件（View），对配套技术不做限定，方便用户与现有技术栈快速整合，降低使用成本,开发者只需专注js业务逻辑,无需专注H5,CSS开发

采用 Flexbox 布局

使用 Flexbox 技术，灵活自如地对齐、收缩、扩展元素，轻松搞定移动页面布局。

基于 React.js

基于风靡社区的 React.js 封装组件，沿袭高性能、可复用、易扩展、一处学习多端编写特性。
